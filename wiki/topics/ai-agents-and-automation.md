# Topic: AI Agents & Automation

AI agents are autonomous systems that pursue multi-step goals using tools (web browsers, code interpreters, APIs, email clients) rather than simply responding to individual prompts. They represent the most significant paradigm shift in AI in 2026.

---

## Core Concept

**Chatbot:** Waits for a question, answers it, waits again.
**Agent:** Receives a goal, plans steps, executes them using tools, reports back.

The shift from "Human-in-the-Loop" (HITL) to "Human-on-the-Loop" (HOTL) — you manage the agent doing the work, rather than doing each step yourself.

## Key Agent Categories (2026)

- **Research agents:** Perplexity AI, Claude with tools, ChatGPT with browsing
- **Coding agents:** Cursor, GitHub Copilot Workspace
- **Productivity agents:** Microsoft Copilot (Teams), Google Workspace AI
- **Automation agents:** Zapier AI, Make
- **No-code chatbot agents:** Botpress, Voiceflow, Typebot

## Current Limitations

- Agents still make mistakes (goal misinterpretation, tool misuse, loops, confident errors)
- "Fully autonomous" agents not ready for high-stakes decisions without oversight
- Best current use: contained, reversible, low-stakes tasks
