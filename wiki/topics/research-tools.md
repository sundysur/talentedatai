# Topic: Research Tools

AI has fundamentally changed how information is researched and verified. In 2026, two tools stand out for research-specific use cases: Perplexity AI (web-based, real-time) and NotebookLM (document-grounded). Both are distinct from general AI assistants.

## Tool Comparison

| | Perplexity AI | Google NotebookLM |
|---|---|---|
| **Data source** | Live web (real-time) | Your uploaded documents only |
| **Citations** | Yes — numbered, clickable | Yes — inline to source |
| **Hallucination risk** | Lower (cited) but still present | Very low (grounded to sources) |
| **Internet required** | Yes | No (after upload) |
| **Free tier** | Unlimited quick searches + 5 Pro/day | Up to 100 notebooks, 50 sources each |
| **Paid** | $20/month (unlimited Pro, model choice) | Free (generous) |
| **Unique feature** | Conversational follow-up search | Audio Overview (podcast from docs) |
| **Best for** | Current facts, web research, live data | Document analysis, study, cross-source Q&A |

## When to Use Each

**Use Perplexity when:**
- You need a factual answer with a verifiable source
- You need current/live information (news, stock prices, recent events)
- You're doing competitor research or market surveys
- You need to fact-check a specific claim before publishing

**Use NotebookLM when:**
- You have a pile of documents (PDFs, transcripts, reports) to understand
- You want to ask questions across multiple documents simultaneously
- You want to generate an Audio Overview for passive learning during commute
- You're preparing for an exam or literature review

## Key Limitation of Both Tools

- **Perplexity:** Source quality is inconsistent; still hallucinates on numerical data — always verify numbers
- **NotebookLM:** No internet access — manually kept sources; doesn't know about events outside your uploads; Audio Overview not customisable
