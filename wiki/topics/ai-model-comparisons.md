## Quick Reference: When to Use Which

| Use Case | Best Tool |
|---|---|
| Long-form writing, nuanced voice | **Claude** |
| Long document analysis (50+ pages) | **Claude** (200K token context) |
| Reasoning & logic problems | **Claude** |
| Code generation (Python, JS, SQL) | **ChatGPT-4o** |
| Image generation (via DALL-E 3) | **ChatGPT-4o** |
| Plugin integrations | **ChatGPT-4o** |
| Real-time web search + citations | **Perplexity AI** |
| Factual Q&A, current events | **Perplexity AI** (or Gemini) |
| Google Workspace integration | **Gemini** |
| Image/chart/handwriting analysis | **Gemini** |
| Document Q&A from uploaded files | **NotebookLM** |

## Key Cross-Article Findings

- **Hallucination:** Claude admits uncertainty more readily; ChatGPT occasionally produces confident wrong answers; Perplexity's citations make errors easier to catch
- **Context windows:** Claude 200K > ChatGPT 128K — decisive for long documents
- **The recommendation across articles:** Start with Claude for most knowledge workers; add ChatGPT for multimedia/plugins; Perplexity for research; Gemini if in Google ecosystem
