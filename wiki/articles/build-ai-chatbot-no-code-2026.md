## Summary

Step-by-step guide to building a working AI chatbot (FAQ bot or lead-capture bot) in under 2 hours using Botpress (free tier) + an Anthropic or OpenAI API key. Six steps: account setup → connect AI brain → write system prompt → add knowledge base → test → deploy/embed. Covers improvements and next steps (CRM integration, ElevenLabs voice, AI agents).

---

## Six Build Steps

1. **Create Botpress account** — Free, no credit card; name bot specifically for purpose clarity
2. **Connect AI brain** — Get Anthropic API key (console.anthropic.com) or OpenAI key; paste into Botpress integrations
3. **Write system prompt** — 4 elements: Tone, Scope, Uncertainty handling, Data collection trigger
4. **Add knowledge base** — 10–20 Q&A pairs outperform a 50-page PDF; treat as living document
5. **Test thoroughly** — 5 test question types: in-KB, out-of-KB, follow-up, off-topic, ambiguous
6. **Deploy/embed** — Copy JS snippet into `</body>` tag; or use Botpress shareable link

## Key Concepts

- **System prompt = job description:** 80% of chatbot quality comes from a specific, well-written system prompt.
- **Knowledge base quality:** Focused Q&A pairs >> large unstructured documents.
- **Test before launch:** 5 question types reveal all common failure modes.
- **Next level:** CRM via Zapier, voice via ElevenLabs, full AI agents.
